# Blueaxis — Under Construction

A premium, accessible coming-soon page for Blueaxis.

## Preview locally

Open `index.html` in a browser, or serve the folder:

```bash
npx serve .
```

## Customize

- **Email & social links** — Edit footer and `mailto` in `index.html`.
- **Progress %** — Change `PROGRESS_TARGET` in `main.js` and milestone copy in `index.html`.
- **Notify form** — Wire `main.js` submit handler to your API or newsletter provider.

## Files

| File         | Purpose                          |
| ------------ | -------------------------------- |
| `index.html` | Semantic structure & content     |
| `styles.css` | Design system & layout           |
| `main.js`    | Progress animation & form UX     |
